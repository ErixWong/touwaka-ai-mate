## AI工具Panel设计

### Panel位置

**方案A：底部固定Panel**（推荐）
- 优点：不遮挡左侧组织树
- 优点：选中清单一目了然
- 缺点：占用底部空间（约150px高度）

**方案B：右侧侧边Panel**
- 优点：不占用底部空间
- 缺点：遮挡部分合同清单

**方案C：自动隐藏Panel**（推荐）
- 默认隐藏
- 选择合同后自动展开
- 点击[x]清空后自动隐藏

---

### Panel内容

**选中清单**：
```
已选择3份合同
├─ 1. ThinkPad供货协议（v1.1）          [×删除]
├─ 2. 联想北京质量保证协议（v1.0）      [×删除]
└─ 3. 华为合作框架协议（v1.0）          [×删除]

[AI解读]  [AI比对]
```

**交互细节**：
- ✅ 每项显示：合同名称 + 当前版本号
- ✅ 每项右侧：[×删除]按钮（点击删除该项）
- ✅ Panel顶部：已选择N份合同
- ✅ 底部按钮：
  - `[AI解读]`（选择1份以上可点击）
  - `[AI比对]`（选择正好2份可点击）

---

### Panel显示逻辑

**自动展开条件**：
- ✅ 选择第1份合同 → Panel展开
- ✅ Panel显示：已选择1份合同

**自动隐藏条件**：
- ✅ 清空所有选择 → Panel隐藏

**按钮禁用逻辑**：
- ✅ 选择0份 → `[AI解读]`禁用，`[AI比对]`禁用
- ✅ 选择1份 → `[AI解读]`可点击，`[AI比对]`禁用
- ✅ 选择2份 → `[AI解读]`可点击，`[AI比对]`可点击
- ✅ 选择3份以上 → `[AI解读]`可点击，`[AI比对]`禁用（仅支持2份比对）

## 合同清单交互

### 合同卡片

```
□ ThinkPad供货协议
  金额：1,200,000元  状态：已批准 ✓
  签署：2026-01-15  过期：2026-12-31
  版本（展开）：v1.0草稿 | v1.1已批准⭐

**交互逻辑**：
- ✅ 点击checkbox → 合同加入选中清单

**展开状态**：
```

└─ v1.1 已批准 ⭐ 2026-01-15  [查看] [加入分析] ← 当前版本

**隐藏状态**：
```
版本（2个）：[展开▼]
```

**交互逻辑**：
- ✅ 点击[展开▼] → 显示版本列表
- ✅ 点击[隐藏▼] → 隐藏版本列表
- ✅ 点击[加入分析] → 该版本加入选中清单

## 前端组件拆分

### ContractListPanel.vue（主界面右侧）

**功能**：
- 合同清单显示（筛选、排序）
- 合同卡片（checkbox + 关键信息）
- 版本展开/隐藏
- 触发选中事件

**Events**：
```typescript
@select(contractId: string, versionId: string)
@unselect(contractId: string, versionId: string)
@click-contract(contractId: string)
@click-version(versionId: string)
```

---

### AIToolsPanel.vue（底部Panel）

**功能**：
- 显示选中清单
- 删除选中项
- AI解读按钮
- AI比对按钮

**Props**：
```typescript
interface AIToolsPanelProps {
  selectedContracts: SelectedContract[];
  visible: boolean;  // 自动隐藏逻辑
}

interface SelectedContract {
  contractId: string;
  contractName: string;
  versionId: string;
  versionNumber: string;
}
```

**Events**：
```typescript
@remove(contractId: string)
@ai-interpret()
@ai-compare()
```

---

### AIInterpretPage.vue（AI解读页面）

**功能**：
- 显示选中合同列表
- AI对话界面（Chat组件）
- 提问 + 回答历史

**Props**：
```typescript
interface AIInterpretPageProps {
  selectedContracts: SelectedContract[];
}
```

---

### AIComparePage.vue（AI比对页面）

**功能**：
- 显示比对合同列表（2份）
- 比对结果（多维度：元数据、条款、章节、相似度）
- 生成比对报告

**Props**：
```typescript
interface AIComparePageProps {
  contract1: SelectedContract;
  contract2: SelectedContract;
}
```

## API设计

### AI解读API

```javascript
// POST /api/contract-v2/ai/interpret
{
  contract_ids: ["contract001", "contract002", "contract003"],
  question: "这三份合同的金额总和是多少？"
}

// 返回
{
  answer: "三份合同的金额总和为3,700,000元...",
  references: [
    { contract_id: "contract001", section: "金额条款" },
    { contract_id: "contract002", section: "金额条款" },
    ...
  ]
}
```

---

### AI比对API

```javascript
// POST /api/contract-v2/ai/compare
{
  contract_id_1: "contract001",
  contract_id_2: "contract002",
  compare_type: "all"  // all/metadata/text/section
}

// 返回
{
  compare_id: "compare001",
  metadata_diff: {...},
  text_diff: "...",
  section_diff: {...},
  similarity_score: 65.8
}
```

## 总结

**简化后的主界面设计**：

**左侧**：组织树导航
**右侧**：合同清单 + AI工具Panel

**AI工具Panel**（底部，自动隐藏）：
- 选中清单（每项可删除）
- [AI解读]按钮（1份以上）
- [AI比对]按钮（正好2份）

**点击按钮进入单独页面**：
- AI解读页面 → AI对话界面
- AI比对页面 → 比对结果界面

**核心优势**：
- ✅ 界面简洁（去掉Dashboard）
- ✅ 交互直观（底部Panel）
- ✅ 功能聚焦（合同清单 + AI工具）
