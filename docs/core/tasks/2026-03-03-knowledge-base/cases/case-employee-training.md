# 案例3：员工培训体系知识库

> 岗位-流程-操作三级结构的培训/SOP 知识库

## 场景描述

某企业需要构建员工培训体系知识库，为每个岗位的每个职位提供：
- 岗位职责说明
- 日常工作流程
- 每个流程的标准作业指导（SOP）
- 培训材料和考核标准

## 知识点示例

### 示例1：岗位职责说明

```markdown
# 销售代表岗位职责说明

> 📍 位置：员工培训体系 > 销售部 > 销售代表
> 📅 更新时间：2026-01-15
> 👤 岗位级别：P2-P4

## 岗位概述

销售代表是公司销售团队的基础岗位，负责公司产品/服务的市场推广和客户开发，完成销售目标。

## 核心职责

| 职责 | 权重 | 说明 |
|------|------|------|
| 客户开发 | 30% | 开发新客户，扩大客户基础 |
| 客户维护 | 25% | 维护现有客户关系，提高复购率 |
| 订单处理 | 20% | 处理客户订单，协调内部资源 |
| 市场调研 | 15% | 收集市场信息，反馈客户需求 |
| 销售报表 | 10% | 完成销售数据统计和汇报 |

## 任职要求

### 学历要求
- 大专及以上学历
- 市场营销、工商管理等相关专业优先

### 技能要求
- 良好的沟通表达能力
- 基本的办公软件操作能力
- 了解公司产品知识

### 经验要求
- P2：应届毕业生或 1 年以下经验
- P3：1-3 年销售经验
- P4：3-5 年销售经验，有带人经验

## 绩效指标

| 指标 | P2 目标 | P3 目标 | P4 目标 |
|------|---------|---------|---------|
| 月销售额 | 10 万 | 20 万 | 35 万 |
| 新客户数 | 3 个/月 | 5 个/月 | 8 个/月 |
| 客户满意度 | ≥ 85% | ≥ 88% | ≥ 90% |

## 职业发展

```

    ▼

    ▼
销售经理 (P7-P8) → 管理 10-20 人团队
    │
    ▼
销售总监 (P9+)

### 示例2：标准作业指导（SOP）

```markdown
# 报销单审核

> 📍 位置：员工培训体系 > 财务部 > 出纳 > 日常流程 > 报销处理
> ⏱️ 预计耗时：5-10 分钟/单
> 🔐 所需权限：财务系统-报销审核

## 流程概述

报销单审核是出纳的日常工作之一，确保报销符合公司财务制度，票据真实有效。

## 前置条件

- [ ] 已登录财务系统
- [ ] 有报销审核权限
- [ ] 了解公司报销制度

## 审核标准

### 1. 票据审核

| 检查项 | 要求 | 不合格处理 |
|--------|------|-----------|
| 发票真伪 | 可在税务局网站验证 | 退回，要求更换 |
| 发票抬头 | 必须是公司全称 | 退回，要求更换 |
| 开票日期 | 报销当月或上月 | 超期需特殊审批 |
| 金额一致 | 发票金额 = 报销金额 | 退回，要求修改 |

### 2. 费用类别审核

| 费用类别 | 限额 | 特殊要求 |
|---------|------|---------|
| 交通费 | 实报实销 | 需附行程说明 |
| 餐饮费 | 100 元/人/餐 | 需注明事由和人数 |
| 住宿费 | 按城市标准 | 需附出差审批单 |
| 办公用品 | 需提前审批 | 需附采购申请单 |

### 3. 附件审核

- [ ] 发票原件扫描件清晰
- [ ] 出差审批单（如适用）
- [ ] 采购申请单（如适用）
- [ ] 其他支持性文件

## 操作步骤

### 步骤1：进入审核界面

1. 登录财务系统
2. 进入【费用管理】→【报销审核】
3. 查看待审核列表

![报销审核界面](/kb-images/expense-audit-main.png)

### 步骤2：逐项审核

1. 点击报销单进入详情
2. 核对报销人信息
3. 逐张核对发票
4. 检查费用类别和金额
5. 查看附件是否齐全

### 步骤3：处理决策

**审核通过**：
1. 点击【通过】按钮
2. 系统自动流转到下一环节

**审核退回**：
1. 点击【退回】按钮
2. 选择退回原因
3. 填写具体说明
4. 提交退回

### 步骤4：特殊情况处理

**金额超限**：
- 需要部门主管加签
- 在系统中发起加签流程

**票据问题**：
- 联系报销人说明情况
- 记录沟通结果

## 常见问题

**Q: 发票抬头写错了怎么办？**
A: 联系商家作废重开，或开具红字发票后重开。

**Q: 跨年发票能报销吗？**
A: 原则上不接受跨年发票，特殊情况需财务总监审批。

**Q: 电子发票怎么审核？**
A: 需要验证电子发票的唯一性，防止重复报销。

## 相关知识点

- 报销制度说明 →
- 发票开具规范 →
- 报销款支付流程 →

## 基础统计函数

### SUM / AVERAGE / COUNT

```excel
=SUM(A1:A10)        ' 求和
=AVERAGE(A1:A10)    ' 平均值
=COUNT(A1:A10)      ' 计数（数字）
=COUNTA(A1:A10)     ' 计数（非空）
```

### 条件统计

```excel
=SUMIF(范围, 条件, [求和范围])
=SUMIF(A:A, ">100")              ' A列大于100的和
=SUMIF(B:B, "销售部", C:C)       ' B列是销售部的，C列求和

=COUNTIF(范围, 条件)
=COUNTIF(A:A, ">=60")            ' A列大于等于60的个数
```

## 文本函数

### 字符串处理

```excel
=LEFT(A1, 3)        ' 取左边3个字符
=RIGHT(A1, 3)       ' 取右边3个字符
=MID(A1, 2, 3)      ' 从第2个字符开始取3个
=LEN(A1)            ' 字符串长度
```

### 文本合并

```excel
=CONCAT(A1, "-", B1)           ' 合并文本
=A1 & " - " & B1               ' 使用 & 合并
=TEXT(A1, "yyyy-mm-dd")        ' 格式化日期
```

## 查找函数

### VLOOKUP

```excel
=VLOOKUP(查找值, 查找范围, 返回列号, [匹配方式])

示例：根据员工编号查找姓名
=VLOOKUP(E1, A:B, 2, FALSE)
```

![VLOOKUP示例](/kb-images/excel-vlookup.png)

### XLOOKUP（新版 Excel）

```excel
=XLOOKUP(查找值, 查找范围, 返回范围)

示例：
=XLOOKUP(E1, A:A, B:B)
```
