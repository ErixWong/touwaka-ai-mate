# 夜间代码分析任务

## 执行时间
每日 02:00 自动执行

## 任务清单

1. **代码分析**
   - 扫描 `src/` 识别未被调用的函数
   - 检测功能重复的 API（参数/逻辑相似度 > 70%）
   - 识别超长函数（> 100 行）和高复杂度函数（圈复杂度 > 15）

2. **文档审计**
   - 对比 `docs/` 与实际实现的一致性
   - 标记过时/错误的描述
   - 列出缺失文档的新功能

3. **生成报告**
   - 输出路径: `docs/reports/nightly/YYYY-MM-DD.md`
   - 如果目录不存在则自动创建

## 执行原则

- **零交互**: 默认接受所有分析结果
- **只读模式**: 不修改任何代码/文档
- **自动归档**: 按日期保存历史报告
- **失败处理**: 记录错误但不中断流程
