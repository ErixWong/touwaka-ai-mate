## 变更说明

为 HTML 文件预览添加源码查看和编辑功能。

### 功能改进

1. **源码/预览切换按钮** - 在 HTML 预览顶部添加切换按钮
   - 点击"源码"（`<>` 图标）切换到源码查看模式
   - 点击"预览"（`👁️` 图标）切换回渲染预览模式

2. **源码高亮显示** - 使用 CodePreview 组件
   - 支持语法高亮
   - 显示行号
   - 支持复制代码

3. **编辑支持** - input 目录下的 HTML 文件
   - 源码模式下可以编辑
   - 支持取消和保存

### 技术实现

- 新增 `showHtmlSource` 状态变量
- 新增 `toggleHtmlSourceMode()` 方法
- 修改 `openPreview()` 方法，打开 HTML 文件时同时加载预览 URL 和源码内容
- 更新 `canEditFile` 计算属性支持 HTML 源码编辑
- 文本/代码预览也统一改用 CodePreview 组件
